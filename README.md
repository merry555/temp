주어진 JSON 데이터를 바탕으로 키워드 순위를 마크다운 표 형식으로 정리해 보겠습니다.

```json
{
  "news": {
    "1주차": ["백신", "코로나", "단일화", "재난지원금", "자치분권"],
    "2주차": ["단일화", "부동산", "코로나", "불법사찰", "여론조사"],
    "3주차": ["부동산", "백신", "단일화", "코로나", "부동산투기"],
    "4주차": ["백신", "코로나", "부동산", "단일화", "자치분권"],
    "5주차": ["부동산", "백신", "재보궐선거", "코로나", "여론조사"]
  }
}
```

키워드 순위는 각 주차 배열의 인덱스 값을 나타냅니다. 예를 들어, `1주차` 배열의 `0`번째 요소는 1위 키워드, `1`번째 요소는 2위 키워드,... `4`번째 요소는 5위 키워드입니다.

다음은 키워드 순위를 나타내는 마크다운 표입니다:

| 주차 | 1위 | 2위 | 3위 | 4위 | 5위 |
|------|------|------|------|------|------|
| 1주차 | 백신 | 코로나 | 단일화 | 재난지원금 | 자치분권 |
| 2주차 | 단일화 | 부동산 | 코로나 | 불법사찰 | 여론조사 |
| 3주차 | 부동산 | 백신 | 단일화 | 코로나 | 부동산투기 |
| 4주차 | 백신 | 코로나 | 부동산 | 단일화 | 자치분권 |
| 5주차 | 부동산 | 백신 | 재보궐선거 | 코로나 | 여론조사 |

이 표는 각 주차의 키워드 순위를 보여줍니다. 각 행은 해당 주차의 키워드 순위를 나타내며, 각 열은 순위(1위, 2위,..., 5위)를 나타냅니다.
